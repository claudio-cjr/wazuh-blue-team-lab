# 📜 Rules — Wazuh Detection Rules

Esta pasta contém regras utilizadas e estudadas durante o desenvolvimento do laboratório Blue Team com Wazuh.

As regras têm como objetivo identificar comportamentos suspeitos, eventos relevantes de segurança e atividades monitoradas no endpoint Windows.

---

# 🎯 Objetivo

O objetivo desta seção é compreender o funcionamento de regras de detecção em um SIEM, incluindo:

* correlação de eventos
* análise de logs
* identificação de comportamentos suspeitos
* monitoramento de processos Windows
* detecção baseada em eventos Sysmon
* estudo de regras do Wazuh

---

# 🧠 Tecnologias Relacionadas

* Wazuh Rules Engine
* Sysmon
* Windows Event Logs
* EventChannel
* MITRE ATT&CK

---

# 📊 Eventos Relacionados

As regras utilizadas no laboratório estão relacionadas a eventos como:

* falhas de autenticação Windows
* execução de PowerShell
* criação de processos
* criação de arquivos suspeitos
* enumeração de usuários locais
* eventos Sysmon

---

# 🛠️ Objetivo Técnico

Os estudos desta pasta ajudam no entendimento de:

* funcionamento do mecanismo de detecção do Wazuh
* parsing de logs Windows
* estrutura de eventos Sysmon
* severidade de alertas
* correlação de eventos em ambiente SOC
